

# 아주대 긱사 어디가 - 구현 계획

## 개요
아주대학교 학생들을 위한 기숙사 정보 및 점수 계산 플랫폼입니다. 아주 블루(#004a98)를 포인트 컬러로 사용한 모던하고 깔끔한 디자인으로 구현합니다.

---

## 페이지 구성

### 1. 랜딩 페이지 (히어로 섹션)
- **"어떤 기숙사에 지원하시나요?"** 메인 타이틀
- 성별 선택 (남/여) → 신분 선택 (학부 신입, 학부 재학, 일반대학원, 외국인, 의대, 간호대, 법학전문대학원) 총 14개 카테고리 버튼
- 글래스모피즘 효과가 적용된 카드 UI
- 아주대 블루 그라데이션 배경의 히어로 섹션

### 2. 지원 가능 기숙사 결과 페이지
- 선택한 카테고리에 따라 지원 가능한 기숙사만 하이라이트
- 지원 불가 기숙사는 흐리게 표시
- 각 기숙사 카드에 태그 표시 (#학부전용, #대학원가능, #신축 등)
- 경쟁률 뱃지 표시 ("신입생 경쟁 치열" 등)

### 3. 점수 계산기 (재학생용)
- **학점 입력**: 슬라이더 또는 직접 입력 → 실시간 학점 점수 변환 (최대 60점)
- **거리/기숙사 포인트**: 기존 거주자 여부 선택 → 거주자면 30점, 신규면 지역 선택으로 거리 점수 계산
- **봉사활동 포인트** (최대 5점), **필수교육 포인트** (최대 5점)
- 실시간 총점 계산 및 표시 (최대 100점)
- Recharts를 활용한 학점-점수 매핑 차트

### 4. 신입생 안내 섹션
- 선발 우선순위 안내: 1순위 (입학성적우수장학금) → 2순위 (거리) → 3순위 (수능점수)
- 거리 점수 기준표 시각화

### 5. 기숙사 비교 카드 섹션
- 남제관, 용지관, 화홍관, 광교관, 국제학사, 일신관 6개 기숙사 정보 카드
- 글래스모피즘 스타일, 입사 가능 대상 태그, 경쟁률 뱃지
- 카드 클릭 시 상세 정보 확인

### 6. 주요 공지사항 요약
- 1년 거주 보장 안내 (1학기 완료 시 2학기 보장)
- 환불 정책 안내 (4주 이내 75% 등)
- 깔끔한 아코디언 또는 카드 형태

## 디자인 & UX
- **포인트 컬러**: 아주 블루 (#004a98) 기반 컬러 시스템
- **글래스모피즘**: 카드 컴포넌트에 반투명 배경 + 블러 효과
- **모바일 반응형**: 학생들이 이동 중 확인할 수 있도록 모바일 최적화
- **스텝 위자드**: 카테고리 선택 → 기숙사 확인 → 점수 계산 단계별 안내
- **언어**: 전체 한국어

- 14개 카테고리별 기숙사 매핑 데이터 내장
